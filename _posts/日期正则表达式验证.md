最近项目中要运用到日期正则表达式验证判断格式是否正确。

1.判断yy-mm-dd
```js
var RegExp= /^(\d{4})(?:(?:-(1[012]|0[1-9]|[1-9]))(?:-(3[01]|[12]\d|0[1-9]|[1-9]))?)?$/;
var date = '2015-11-24 '

RegExp.test(date)
```

2.判断yy-mm-dd hh:mm
```js
var RegExp =  /^[1-2][0-9][0-9][0-9]-([1][0-2]|0?[1-9])-([12][0-9]|3[01]|0?[1-9]) ([01]?[0-9]|[2][0-3]):[0-5]?[0-9]$/ ;

var date = '2015-11-24 8:08'

RegExp.test(date)

```
